StrideSense
GPS-Powered Fitness Tracking  ·  Flutter Mobile App
Built with Flutter Powered by Supabase Android & iOS Dart & PostgreSQL
Case Study
The Problem
We Solved
Runners in our community trained alone with no feedback loop. Progress was invisible.
No affordable wearable. No unified platform. No peer accountability to keep them consistent.
Motivation faded fast without data, structure, or a sense of community pushing them forward.
Target audience: University student athletes running regularly but lacking structured tools, coaching data, or peer accountability.
Pain Points
Three gaps we identified
No GPS Tracking
No way to measure distance, pace or route in real time
✕
No Leaderboards
No competitive ranking to compare performance with peers
✕
No Club System
No way to form groups, join clubs, or run shared challenges
✕
StrideSense fixes all three — in one Flutter app
Project Requirements
What the App Had to Do
Functional Requirements
1
Register & login with email and password
2
Record runs with live GPS distance & timer
3
GPS fallback to simulated route when unavailable
4
Full workout history view with summaries
5
View & edit profile including avatar upload
6
Browse and join running clubs
7
View, join, and track fitness challenges
8
Leaderboard ranked by weekly / monthly distance
9
Secure logout from any device
Non-Functional Requirements
1
Offline-first with local cache & background sync
2
Encrypted auth token storage (flutter_secure_storage)
3
Under 300 ms for all local data operations
4
Location access only with explicit user permission
5
Row Level Security on every database table
6
Android API 21+ and iOS 14+ support
7
Avatar uploads namespaced per user — no collision

User Guide
How to Use StrideSense
Sign Up
→ Tap Get Started on splash screen
→ Fill email, password, name & phone
→ Supabase creates profile automatically
→ Lands on Home dashboard
Log In
→ Enter email and password
→ Session stored securely on device
→ Profile and dashboard loaded from cache
→ Background sync updates fresh data
Record a Workout
→ Open Record tab → tap Start
→ Grant location permission if prompted
→ GPS tracks distance & time live
→ Pause / Stop → choose category → Save
View Progress
→ Home — total distance & weekly pace
→ Profile — weekly distance bar chart
→ Tap any workout for route map & summary
Community
→ Leaderboard — weekly/monthly rankings
→ Browse Challenges → tap to join
→ Browse Clubs → join or create a club
Edit Profile
→ Edit Profile — public display info
→ Personal Info — private email & phone
→ Change Photo — camera or gallery
→ Saves locally first, then auto-syncs
